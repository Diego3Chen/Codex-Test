# Customizing and Deploying Your Personal Resume Website

## Customization Instructions
1. **Clone the Repository:**  
   Open your terminal and run:
   ```bash
   git clone https://github.com/Diego3Chen/Codex-Test.git
   cd Codex-Test
   ```

2. **Modify Content:**  
   Open the `index.html` file in your favorite text editor. Update the text, images, and links to reflect your personal information and resume details.
   
3. **Customize Styles:**  
   If you want to change the look of your website, modify the `styles.css` file to update colors, fonts, and layout according to your preference.

## Deployment Instructions
1. **Commit Your Changes:**  
   After making your changes, add and commit them:
   ```bash
   git add .
   git commit -m "Customize resume website"
   ```

2. **Push Changes to GitHub:**  
   Push your changes back to the repository:
   ```bash
   git push origin main
   ```

3. **Enable GitHub Pages:**  
   - Go to your repository on GitHub.
   - Click on `Settings`.
   - Scroll down to the `Pages` section.
   - Select `main` as the branch to deploy from and click `Save`.

4. **Access Your Website:**  
   Your website will be live at `https://<your-username>.github.io/Codex-Test` within a few minutes after enabling GitHub Pages.

## Additional Tips
- Make sure to keep your repository updated with your latest resume changes. 
- Check out the [GitHub Pages documentation](https://pages.github.com/) for more options and configurations.

Happy coding!
